## **Overview**

This is a Command Line Interface (CLI) Application based in Java that streamlines how students, company representatives and school career centre staff interact during NTU's internship matching process.

<br>

## **User Roles**

Students can:
- Browse available internship opportunities
- Submit internship requests
- View or withdraw previously submitted requests

Company Representatives can:
- Create internship opportunities
- Edit details of pending internship opportunities
- View student requests for opportunities from their company
- Approve or reject requests

Career Staff oversees the entire system by:
- Viewing all opportunities
- Approving creation of company rep accounts
- Approving internship opportunity creation requests
- Approving withdrawal requests

<br>

## **Set-Up**
### Pre-Requisites

Ensure you have Java installed and it is at least Java 17 or a later version. Check your version by 
  ```sh
  java --version
  ```
or install it at [https://www.java.com/en/download/](https://www.java.com/en/download/)

### Getting Started

_Below is how to start our application._

1. Clone the repo
   ```sh
   git clone https://github.com/qorkjnxdk/SC2002_Project.git
   ```
2. Allow Javac to Compile using the app.java entrypoint
   ```sh
   javac App.java
   ```
3. Run the application
   ```sh
   java App.java
   ```
4. Follow prompts on the CLI menus

_Below is how to manually edit/add/delete any data._

1. Access the CSV files in /data
2. Do not change any CSV file names
3. Edit/Delete/Add data based on the schema in each CSV file
